# miniguia-de-educacao-financeira-notebooklm
Projeto com NotebookLM e IA Generativa sobre a baixa educação financeira na era digital. Voltado para jovens e iniciantes, organiza fontes confiáveis para criar um miniguia com resumos, glossário e prompts reutilizáveis, aplicando pensamento crítico, curadoria de informações e aprendizagem ativa com IA.
